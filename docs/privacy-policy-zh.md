# Code Remote 隐私政策

生效日期：2026-05-28

Code Remote 是一个面向开发者的鸿蒙远程客户端，用于连接用户自行安装和配对的桌面端服务 `code-desktop-service`。本应用仅支持在本地局域网或用户可直接访问的本地网络环境内连接用户自行配对的 Mac 或 Windows 桌面服务，不提供公网中转服务，不提供云端 AI 模型服务，不托管用户代码，不内置广告、统计或支付 SDK。

## 1. 我们处理哪些信息

为完成扫码配对、会话同步和文件预览，本应用会在用户设备本地处理以下信息：

- 配对信息：桌面端名称、桌面端服务地址、候选服务地址、证书指纹、公钥哈希、授权 token、配对时间和授权有效期。
- 连接信息：网络连接状态、桌面端在线状态、最近连接时间和连接错误提示。
- 会话信息：会话标题、会话状态、用户输入、助手输出、审批请求、计划进度、文件引用、网页链接和本地预览状态。
- 用户选择的文件或图片：仅在用户主动选择上传、预览、保存或分享时处理。
- 本地缓存：为提升打开速度，本应用可能缓存会话详情、附件预览、图片、文档和网页预览结果。

本应用不会主动读取通讯录、短信、位置、麦克风、日历等与功能无关的信息。

## 2. 权限使用说明

- 网络权限：用于在本地局域网或用户可直接访问的本地网络环境内连接用户自行配对的桌面服务，进行会话同步、消息发送、文件传输和本地网页预览。
- 网络状态权限：用于判断网络连接状态，帮助用户发现和重连桌面服务。
- 相机权限：仅用于扫描桌面端管理页上的配对二维码，不拍照、不录像、不保存相机画面。
- 震动权限：用于配对、发送、审批等操作的轻量触感反馈。

本版本不申请图库写入受限权限。图片保存通过 HarmonyOS 系统保存控件或授权弹窗写入图库；其他文件保存通过系统文件保存面板完成，用户需要在对应系统界面中确认保存。

## 3. 信息如何存储

配对凭据和部分运行缓存保存在用户的鸿蒙设备本地。配对凭据使用系统安全存储能力保存；会话和附件缓存保存在应用本地目录中。用户可以在应用设置页取消配对、清理缓存，也可以在系统设置中撤销权限或卸载应用。

## 4. 信息如何传输

用户完成扫码配对后，本应用会通过 HTTPS/WebSocket 与用户自行配对的桌面端服务通信。会话内容、用户输入、附件、图片和网页预览链接可能在鸿蒙设备与该桌面端服务之间传输。

本应用开发者不提供公网中转服务器，不提供云端代码托管服务，也不会主动收集、上传或存储用户代码、会话内容和附件。用户的会话内容、指令、文件列表和附件仅在用户的鸿蒙设备与用户自行配对的桌面服务之间传输。

如果用户在桌面端配置了 Codex、Claude Code、openclaw、hermes 或其他 AI 编程工具，这些工具可能根据用户的操作将相关内容发送到其对应服务。该部分处理行为由用户选择和配置的第三方工具及其服务条款、隐私政策约束。

## 5. 第三方 SDK 和共享

本应用当前不内置广告 SDK、统计 SDK、支付 SDK或社交登录 SDK。除用户自行配对的桌面端服务和用户主动配置的第三方 AI 工具外，本应用开发者不建设云端服务器接收用户会话内容或代码内容，不提供公网中转、云端同步或云端代码托管服务。

## 6. 用户权利

用户可以：

- 取消桌面端配对。
- 重新扫码授权，刷新授权有效期。
- 清理本地缓存。
- 在系统设置中撤销相机等权限。
- 删除通过系统保存控件、授权弹窗或系统保存面板保存到本机的文件。
- 通过 GitHub Issues 或应用市场开发者联系方式反馈隐私相关问题。

## 7. 未成年人保护

本应用面向具备开发工具使用能力的开发者，不以未成年人为主要服务对象。未成年人使用本应用前，应取得监护人同意并在监护人指导下使用。

- 应用市场开发者联系方式：以上架后台填写的信息为准

## 9. 政策更新

当应用功能、权限、数据处理方式发生重要变化时，我们会更新本隐私政策，并在应用发布说明或应用市场页面中提示。
